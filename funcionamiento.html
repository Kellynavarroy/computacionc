Mecanismo de funcionamiento

Método

Este sistema se basa en utilizar las leyes de mecánica cuántica, es decir se apoyan de la física que afirma que los objetos pueden estar en dos lugares al mismo tiempo.

Construcción

Lo primero es lograr construir un dispositivo a pequeña escala. Para construir una computadora a gran escala se debe esamblar a un plano que trabaja con átomos cargados o iones que se capturan para que funcionen como qubits y operen a temperatura ambiente.

Circuito electrónico

La base de estos circuitos son las compuertas lógicas, las cuales son utilizadas para realizar las computaciones. Se alinea un rayo láser individual sobre cada ión, es un poco complocado ya que hablamos de una cantidad enorme de qubits.
Como una opción innovadora se planea utilizar microondas y voltajes para operar estas compuestas. Pero la solución más llamativa es utilizar campos eléctricos para empujar los iones de un módulo a otro.

Lenguaje de programación

Ahora bien hablemos de su lenguaje de programación. Si bien es cierto que en computación cuántica se carece de un lenguaje para programar que sea propio, se toma como opción desarrollar e implementar algoritmos específicos, pero como sabemos la informática tradicional cuenta con lenguajes estandarizados como Java, SQL y Python.
La buena noticia es que científicos de ETH Zurich, diseñaron el primer lenguaje de programación para computadoras cuánticas y se llama Silq. Para saber más, pulsar el botón, el se cree invisible, solo ponga el cursor encima y listo. Página web de Silq
